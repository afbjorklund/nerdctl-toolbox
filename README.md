# nerdctl-toolbox

pocket-protector, a.k.a. nerdctl-toolbox (or `nerdbox`)

A tool, to avoid having to install software on the host.

## Usage

`nerdbox build`
`nerdbox create`
`nerdbox shell`
`nerdbox remove`

`nerdbox list`
`nerdbox run`
